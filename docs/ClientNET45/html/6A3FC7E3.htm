Stormancer.Networking Namespace
The Stormancer.Networking namespace contains interfaces used internally by the network engine. They can be used to create wrappers to enable advanced logging and message interception scenarios.
Classes

PacketProcessorConfig
Contains method to register handlers for message types when passed to the IPacketProcessor.RegisterProcessor method.
