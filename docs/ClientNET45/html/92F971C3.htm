Stormancer Namespace

[This is preliminary documentation and is subject to change.]

This namespace contains the main SDK class and interfaces necessary to build applications that interact with Stormancer servers.
The Stormancer client SDK for .NET enables developpers to create games and applications connecting to Stormancer servers in realtime in UDP. It provides easy to use high level abstractions as well as low level API. For more information, see Stormancer online documentation
